# Stamina

`RESOURCE_STAMINA`

> Physical endurance.

## What it does

Physical endurance.

**Color:** Blue
**Default Regen:** (level*2) + 3 / 1000ms

## Valid values

- **Usage:** Sprinting, Wall-running, Acrobatics, Dodge.
- **Usage:** Clone/ARC Trooper, Hero.

## Notes

- Regenerates quickly when walking/standing.
