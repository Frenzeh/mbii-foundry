# Fuel

`RESOURCE_FUEL`

> Propellant for jetpacks and flamethrowers.

## What it does

Propellant for jetpacks and flamethrowers.

**Color:** Yellow
**Default Regen:** level / 1000ms

## Valid values

- **Usage:** Flying, Flamethrower, Wrist Laser.
- **Regen:** Regenerates slowly when grounded.

## Notes

- Standard for Mandalorians.
