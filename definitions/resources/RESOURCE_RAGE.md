# Rage / Fury

`RESOURCE_RAGE`

> Adrenaline and anger.

## What it does

Adrenaline and anger.

**Color:** Red
**Default Regen:** 1 / 1000ms

## Valid values

- **Build:** Builds up by **dealing damage** (if class has `MB_ATT_WOOKIEE_FURY`).
- **Usage:** Activating Wookiee Fury.

## Notes

- Used by Wookiees to trigger their berserk state.
