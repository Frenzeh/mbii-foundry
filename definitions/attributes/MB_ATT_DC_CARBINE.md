# DC-15S Carbine

`MB_ATT_DC_CARBINE`

> Compact Clone carbine attribute. Fast primary, heavy secondary.

## What it does

Grants `WP_DC_CARBINE` — the shorter, lighter counterpart to the DC-15A rifle. Fast 200ms primary cadence with the standard blaster ammo pool, and a slow but high-energy secondary that fires a heavier shot.

## Per level

- **Level 1** — Basic carbine; standard rifle-tier damage.
- **Level 2** — Heavy secondary unlocked.
- **Level 3** — Maximum damage; tighter spread.

## Notes

- Unlocks `WP_DC_CARBINE`.
- Uses `AMMO_BLASTER` shared with the E-11 — generous reserves.
- Common on Clone Trooper / Sergeant kits.
