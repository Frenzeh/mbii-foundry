# Remote Detonate

`MB_ATT_REMOTE_DETONATE`

> Remote-trigger ability for placed explosives. Detonates on command rather than timer.

## What it does

Adds remote-trigger functionality to placed explosives — det packs, sticky bombs, and similar can be detonated on command via the Special key rather than waiting for proximity or a timer. Critical for ambush setups where exact timing matters.

## Per level

- **Level 1** — Manual remote detonation enabled for det packs.
- **Level 2** — Extended to sticky bombs and other placeables.
- **Level 3** — Faster detonation response; range-extension.

## Notes

- Pairs with `MB_ATT_DET_PACK`, `MB_ATT_STICKY_BOMBS`, `MB_ATT_TRIP_MINES`.
- Critical for breaching/ambush playstyles.
- Engineer, Demolition, Saboteur loadouts.
