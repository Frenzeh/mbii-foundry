# MB Points

`mbPoints`

> The total points available for a Custom Build character to spend.

## What it does

The total points available for a Custom Build character to spend.

**Default:** 0

## Valid values

- **Pool:** The budget the player has to buy ranks in `c_att_skill_*`.

## Notes

- Standard Legends value: ~100-150 depending on power level.
