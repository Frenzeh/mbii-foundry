# Medpac

`EAS_HI_MEDPAC`

> Uses a Bacta canister to restore health.

## What it does

Plays the medpac animation and restores a chunk of HP up to the user's max. Same effect as the holdable `HI_MEDPAC` but accessible from a special-key binding so the player doesn't have to scroll the inventory wheel.

## Notes

- Charges share the underlying Bacta inventory — bound key and item slot both deplete the same counter.
- Cannot be cancelled mid-use; weapon is locked out for the heal animation.
- Use `EAS_HI_MEDPAC_BIG` for the larger "Big Bacta" variant.
