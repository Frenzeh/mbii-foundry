# MBII Foundry Roadmap

## 🚀 Current Focus: Stability & Safety
*   [ ] **Undo/Redo System**: Implement Command pattern for safe editing.
*   [ ] **Dirty State Tracking**: Visual indicators (*) for unsaved files.
*   [ ] **Safe Parsing**: Improve file parsing to prevent data loss (comments/formatting).

## 📅 Short Term (The "Workflow" Update)
*   [ ] **PK3 Builder**: One-click button to save, pack, and install the character for testing.
*   [ ] **Log Watcher**: Integrated terminal showing `games.log` errors in real-time.
*   [ ] **Validator Integration**: Real-time red borders/tooltips for invalid values (instead of just on save).

## 🔮 Long Term (The "Pro" Update)
*   [ ] **Live Source Editing**: Bi-directional editing in the "Source" tab (Code -> UI).
*   [ ] **Visual Saber Designer**: Real-time preview of saber blades (color, length, glow).
*   [ ] **Model Viewer Integration**: Embed a lightweight 3D view or robust screenshots for the Asset Browser.
*   [ ] **Wiki Integration**: Fetch mechanic formulas (AP/BP calculations) directly from the MBII Wiki.

## 🧊 Icebox (Ideas)
*   **Discord Integration**: "Share my build" button.
*   **Template Library**: Cloud-hosted library of starter templates.
*   **Dark Mode Toggle**: (Already partially supported by theme, needs explicit toggle).
