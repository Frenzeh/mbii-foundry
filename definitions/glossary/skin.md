# Skin

`skin`

> The texture set applied to the model.

## What it does

The texture set applied to the model.

**Format:** `skin_name` (e.g., `default`, `red`, `blue`, `rgb`)

## Notes

- Must exist within the model's `.skin` files.
- `default` is the most common.
- `rgb` allows for custom colors if the model supports it.
