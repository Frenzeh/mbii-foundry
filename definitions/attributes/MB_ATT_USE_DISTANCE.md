# Use Distance

`MB_ATT_USE_DISTANCE`

> Increases interaction range for objectives, dispensers, and item pickups.

## What it does

Extends the maximum distance at which the user can press Use to interact — objectives, doors, ally dispense actions, item pickups. Useful for objective-runner builds where reaching from cover is preferable to fully exposing.

## Per level

- **Level 1** — slight range bonus.
- **Level 2** — moderate range bonus.
- **Level 3** — significantly extended range.

## Notes

- Also extends the range of all dispenser actions (Health, Ammo, Armor, Force, BP).
- Categorized as Advanced — utility tuning, rarely a primary pick.
- Stacks well with `MB_ATT_HACK_MULTIPLIER` for dedicated objective-runners.
