# Trooper

`MB_CLASS_TROOPER`

> Rebel frontline grunt — the red-team mirror of Soldier.

## Role

The Rebellion's basic rifle infantry. Statline is identical to Soldier (50 HP, 187.5 speed, `AP_5_15_25_35_M35`). Loadout flavour differs in community specs — Trooper tends toward blaster rifle / A280 / DLT-20A configurations where Soldier leans E-11 / repeater. Cheap, specializable, expected filler on any Rebel team.

## Default loadout

- **Base health** — 50
- **Base speed** — 187.5
- **Armor pool** — `AP_5_15_25_35_M35`
- **Melee moves** — Punch only (`MM_PUNCH`)
- **Resource** — none
- **Abilities (CS1-4)** — none by default
- **Forcecfg** — `forcecfg/red/soldier`
- **Model** — `rebel`

## Signature mechanics

- Same `MB_ATT_CCTRAINING` scaling as Soldier.
- Swaps easily into sniper roles via `MB_ATT_DLT20A` or ranged support via `MB_ATT_A280`.

## Counters

- Identical weaknesses to Soldier — melee, sabers, Wookiees all crush a vanilla Trooper.
