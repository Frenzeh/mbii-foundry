# Force Focus

`HELD_FORCEFOCUS`

> Hits restore Force Pool to the firer.

## What it does

Each successful hit returns Force Pool (FP) to the user — turning gunplay into a force-regen tool for hybrid classes.

## Notes

- Added in R22.0.00.
- Designed for force-using gunner archetypes.
- Amount returned per hit is balanced against weapon's base damage.
