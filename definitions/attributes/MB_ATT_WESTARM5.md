# Westar M5

`MB_ATT_WESTARM5`

Heavy ARC Trooper Rifle. Modular and powerful.

> **Default:** None.

### Stats
* **Primary Damage:** **30**
* **Sniper Damage:** **69**
* **Velocity:** 4600 (Primary) / 9825 (Sniper)
* **Ammo Cost:** 2 (Primary) / ~12 (Sniper)

### Levels

* **Level 1**:
 * **Standard**. Automatic fire.
 * **Scope:** Can attach scope for sniper mode.

* **Level 2**:
 * **Improved**. Higher accuracy.
 * **Sprint:** Can fire while sprinting (if Stamina 2+).

### Tips
* The **Sniper Mode** hits extremely hard (69 damage), capable of 2-shotting many classes.
* Requires `MB_ATT_ARC_RIFLE_SCOPE` to use the sniper mode.
