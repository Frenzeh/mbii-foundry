# Final Objective

`final`

> Marks this objective as the game-ending condition.

## What it does

Marks this objective as the game-ending condition.

## Valid values

- **1:** Completing this objective ends the round immediately (Attacker Win).
- **0:** Intermediate objective. Unlocks next stage or extends time.

## Notes

- Only one objective per team usually has this (or the last one in a sequence).
