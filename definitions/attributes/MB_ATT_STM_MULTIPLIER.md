# Stamina Multiplier

`MB_ATT_STM_MULTIPLIER`

> Multiplier for the Stamina pool size.

## What it does

Scales the maximum Stamina pool. Used on Sprint-heavy classes (ARC, Hero) to give them a deeper reserve without changing the regen rate. Default 1.0.

## Notes

- Pool affects sprint duration, wall-runs, and acrobatic actions tied to `MB_ATT_DEXTERITY`.
- Doesn't change regen rate — pair with `MB_ATT_RESOURCE_REGEN_AMOUNT` for full tuning.
- Cosmetic effect only on classes that don't use Stamina.
