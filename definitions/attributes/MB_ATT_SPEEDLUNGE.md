# Speed Lunge

`MB_ATT_SPEEDLUNGE`

> Enables saber speed-lunge without the Force Speed 3 + Jump 3 prereq.

## What it does

Standalone unlock for the saber speed-lunge — a fast horizontal dash-attack that vanilla MBII gates behind both Force Speed 3 and Jump 3. With this attribute, classes can be given the lunge without the full Force loadout.

## Per level

- **Level 1** — basic speed-lunge unlocked.
- **Level 2** — improved range.
- **Level 3** — full damage and longest reach.

## Notes

- Useful for non-Jedi classes that should still have an aggressive opener (Mandalorian saberist, Darksider builds).
- Replaces the Force-Speed 3 / Jump 3 dual prereq with a single attribute pick.
