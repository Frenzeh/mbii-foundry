# Close Combat Training

`MB_ATT_CCTRAINING`

Specialized CQC (Close Quarters Combat) drills for soldiers.

> **Class Specific:** Soldier.

### Levels

* **Rank 1:**
 * **Speed:** **1.2x** movement speed while using Melee/Fists.
* **Rank 2:**
 * **Speed (Melee):** **1.2x**.
 * **Speed (Gun):** **1.1x** movement speed with weapons.
 * **Getup:** Improved Getup animation speed (1 -> 2).
* **Rank 3:**
 * **Speed:** **1.2x** all-around movement speed (Weapon or Melee).
 * **Getup:** Improved Getup animation speed.

### Usage
* Turns a standard Soldier into a mobile skirmisher.
* Allows soldiers to outrun heavy classes.
