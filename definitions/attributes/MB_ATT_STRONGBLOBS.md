# Concussion Blobs

`MB_ATT_STRONGBLOBS`

High-impact concussion rounds for the DC-15A Clone Rifle.

**Class Specific:** Clone Trooper, ARC Trooper.

### Levels

* **Level 1**:
 * 3 Concussion rounds.
 * Large explosion radius.
 * Knocks down enemies in a wide area.

* **Level 2**:
 * 6 Concussion rounds.

* **Level 3**:
 * 9 Concussion rounds.

### Notes
* Requires `MB_ATT_CLONERIFLE`.
* Cannot be equipped simultaneously with Clone Blobs (Ion) on some server configs, but FA allows overrides.
