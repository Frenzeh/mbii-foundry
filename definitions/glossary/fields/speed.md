# Speed Multiplier

`speed`

Multiplies the class's default movement speed.

> **Default:** 1.0

### Mechanics
* **1.0:** Normal class speed.
* **1.1:** 10% Faster.
* **0.9:** 10% Slower.

### Tips
* Jedi Base: ~250?
* Soldier Base: ~200?
* Use `baseSpeed` for absolute control, `speed` for relative scaling.
