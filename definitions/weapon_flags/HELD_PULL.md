# Pulls

`HELD_PULL`

> Drags hit target toward the firer.

## What it does

Hits apply a vector toward the firer, effectively yanking the target closer. Functional inverse of `HELD_KNOCKBACK`.

## Notes

- Added in R22.0.00.
- Useful for "harpoon" / "tractor" weapon concepts.
- Range and pull strength scale with weapon's base knockback.
