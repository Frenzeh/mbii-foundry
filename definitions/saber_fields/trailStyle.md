# Trail Style

`trailStyle`

> Changes the visual trail left by the lightsaber blade.

## What it does

Changes the visual trail left by the lightsaber blade.

**Default:** 0 (Standard Blur)

## Valid values

- **0:** Standard Motion Blur.
- **1:** Solid Ribbon.
- **2:** Sparkles/Particles (Unstable blade effect).

## Notes

- Use **2** for Kylo Ren style "crackle" blades.
