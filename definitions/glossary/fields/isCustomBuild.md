# Custom Build

`isCustomBuild`

> Enables the Point Buy system for this character.

## What it does

Enables the Point Buy system for this character.

**Default:** 0 (Disabled)

## Valid values

- **1 (Enabled):** The character starts with a pool of `mbPoints` and must purchase attributes/weapons in the UI.
- **0 (Disabled):** The character spawns with the preset `attributes` and `weapons` defined in the file.

## Notes

- Required for "Legends" style configurable classes.
