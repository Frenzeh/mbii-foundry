# Dextrous Movement

`MB_ATT_DEXTERITY`

Increases base movement speed and acrobatic capabilities.

> **Default:** 0 (Normal human speed).

### Levels

* **Rank 1:**
 * **Speed:** Slightly increased run speed.
 * **Acrobatics:** Basic rolls (Forward/Back/Side + Crouch).
 * **Wall Run:** Short wall runs.
* **Rank 2:**
 * **Speed:** Faster run speed.
 * **Wall Run:** Longer duration.
 * **Recovery:** "Getup Rolls" (recover quickly from knockdowns).
* **Rank 3:**
 * **Speed:** Maximum run speed.
 * **Wall Run:** Infinite duration (stamina permitting).
 * **Recovery:** "Quick Getup" (instant recovery from knockdowns).
 * **Advanced Acros:** Flips, kicks (if class supports).

### Tips
* Crucial for survival against lightsaber users.
* Combine with `MB_ATT_STAMINA` for extended mobility.
