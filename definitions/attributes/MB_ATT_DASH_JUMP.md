# Dash Jump

`MB_ATT_DASH_JUMP`

> Allows jumping at any point during a Dash.

## What it does

Decouples the Dash animation lockout from the jump command — the user can press jump mid-dash to convert horizontal momentum into a vertical leap. Used for high-mobility flanker classes.

## Per level

- **Level 1** — late-dash jump only.
- **Level 2** — mid-dash jump enabled.
- **Level 3** — jump-cancel from any frame.

## Notes

- Requires `MB_ATT_DASH` to function.
- Pairs with `MB_ATT_BUNNY_HOP` and `MB_ATT_GRAPPLE_HOP` for full chained-mobility builds.
