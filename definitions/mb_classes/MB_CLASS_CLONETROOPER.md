# Clone Trooper

`MB_CLASS_CLONETROOPER`

> Republic infantry — Stamina-driven sprint, DC-15A rifle with alt-fire blob ammunition.

## Role

The core Republic infantry class. Moderate HP (80), reasonable armor pool (10/20/50/80), and the Stamina resource that powers Sprint. Default CS1 is Sprint — holding it consumes Stamina to move faster. Clone Blobs / Strong Blobs (Ion and Concussion respectively) are the signature right-click modes of the DC-15A.

## Default loadout

- **Base health** — 80
- **Base speed** — 187.5
- **Armor pool** — `AP_10_20_50_80_M80`
- **Melee moves** — All
- **Resource** — Stamina (fuels Sprint)
- **Abilities (CS1-4)** — `EAS_HI_SPRINT`, none, none, none
- **Forcecfg** — `forcecfg/red/clone`
- **Model** — `clonetrooper_p1` (skin `rgb` for team coloring)

## Signature mechanics

- **Clone-only attributes** — `MB_ATT_CLONERIFLE` (DC-15A), `MB_ATT_CLONE_PISTOL` (DC-17), `MB_ATT_CLONEBLOBS` (Ion alt-fire), `MB_ATT_STRONGBLOBS` (concussion alt-fire), `MB_ATT_CCTRAINING`.
- **Sprint** burns Stamina for a speed boost; stacks additively with CCTRAINING speed multipliers.
- **Clone Blobs** gate on DC-15A ownership — 3/6/9 ion rounds per rank; high damage vs droids and shields.
- **Strong Blobs** provide a concussive AoE alt-fire in 3/6/9 rounds per rank.

## Counters

- Sith / Jedi in saber range.
- Wookiees once they close.
- Bounty Hunter poison / Sith drain bypass the armor pool.
