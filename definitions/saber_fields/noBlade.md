# No Blade

`noBlade`

Hides the energy blade completely.

### Mechanics
* **1 (True):** Only the hilt model is visible.
* **Usage:** Used when the "hilt" model actually includes the blade geometry (e.g. a physical sword model).

### Tips
* If you are making a Vibroblade and the model includes the metal blade, set `noBlade 1`.
