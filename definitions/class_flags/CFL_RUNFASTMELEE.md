# 馃弮 Run Fast Melee

`CFL_RUNFASTMELEE`

Increases movement speed while holding Melee.

### 鈿欙笍 Mechanics
*   **Speed:** **1.15x** (15% Faster).
*   **Condition:** Active only when `WP_MELEE` is held.

### 馃挕 Tips
*   **Obsolete:** largely replaced by `HELD_SPEED` on weapon flags, but still valid for simple configs.
