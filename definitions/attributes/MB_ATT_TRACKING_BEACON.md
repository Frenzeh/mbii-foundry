# Tracking Beacon

`MB_ATT_TRACKING_BEACON`

> See enemies darted by allied tracking darts.

## What it does

Receiver-side attribute for the Tracking Dart system. If allies have `MB_ATT_TRACKING_DART` and tag enemies, only players with `MB_ATT_TRACKING_BEACON` see the resulting radar pings. Lets a team designate scouts and snipers.

## Per level

- **Level 1** — see darted enemies in close range.
- **Level 2** — extended range.
- **Level 3** — see all darted enemies map-wide.

## Notes

- Categorized as Advanced.
- Pairs with `MB_ATT_TRACKING_DART` (the firing side) and `MB_ATT_RADAR` (broader awareness).
- Useful for sniper / spotter roles.
