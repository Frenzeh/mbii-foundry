# Anti Mind Trick

`MB_ATT_ANTI_MT`

> Resistance to Force Mind Trick (invisibility).

## What it does

Lets the user see through the Force Mind Trick effect. Mind-Tricked enemies still appear (partially or fully) instead of vanishing, breaking the Jedi's primary stealth-flank tool.

## Per level

- **Level 1** — Mind-Tricked enemies appear as **shimmering ghosts** (partial visibility).
- **Level 2** — Mind-Tricked enemies are **fully visible** (immune to the effect).

## Notes

- Categorized as Advanced.
- Critical for sentry classes (SBD, Droideka) defending fixed positions against Jedi/Sith infiltrators.
- Does not break the Mind Trick effect for teammates — only the user sees through it.
