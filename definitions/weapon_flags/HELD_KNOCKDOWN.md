# Knockdown

`HELD_KNOCKDOWN`

> Projectiles knock targets to the ground.

## What it does

Projectiles knock targets to the ground.

## Valid values

- **Effect:** Forces target into knockdown state on hit.
- **Grenades:** If used on Frag Nades, alt throws cause radius knockdown without direct hit.

## Notes

- Very powerful crowd control.
