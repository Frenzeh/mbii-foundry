# 🛡️ Rank Armor

`rankArmor`

Defines the Armor values for Point Buy ranks.

> **Format:** `rank1,rank2,rank3` (e.g., `50,100,150`)

### ⚙️ Mechanics
*   Sets the exact Armor value per purchased rank.

### 💡 Tips
*   Useful for classes where you want Armor upgrades to be massive (e.g. SBD plating).
