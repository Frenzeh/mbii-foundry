# 📐 Model Scale

`modelscale`

Scales the size of the player model and hitbox.

> **Default:** 1.0

### ⚙️ Mechanics
*   **< 1.0:** Smaller (e.g., Yoda = ~0.6-0.7). Harder to hit, can fit in small vents.
*   **> 1.0:** Larger (e.g., Desann = ~1.12). Easier to hit, longer reach.

### 💡 Tips
*   Overrides class defaults (like Wookiee/SBD scaling).
*   Not currently compatible with Droidekas.
