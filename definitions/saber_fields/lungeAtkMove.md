# Lunge Attack

`lungeAtkMove`

> Overrides the animation for the forward lunge attack (Crouch + Forward + Attack).

## What it does

Overrides the animation for the forward lunge attack (Crouch + Forward + Attack).

**Default:** `LS_LUNGE`

## Valid values

- **Value:** Enum string.
- **Gameplay:** Can change a stab into a slash or spin.

## Notes

- `LS_STAFF_SOULCAL` creates a unique spinning lunge.
