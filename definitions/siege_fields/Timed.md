# Time Limit

`Timed`

> Round time limit in seconds.

## What it does

Round time limit in seconds.

**Default:** 0 (Unlimited?)

## Valid values

- **Value:** Seconds (e.g., `1200` = 20 Minutes).
- **Effect:** If time runs out, the Defense team usually wins.

## Notes

- Standard Siege rounds are 15-20 minutes.
