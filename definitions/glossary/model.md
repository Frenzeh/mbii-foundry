# Model

`model`

> The 3D character model used in-game.

## What it does

The 3D character model used in-game.

**Format:** `model_name`

## Valid values

- `jedi_hm` (Human Male Jedi)
- `clonetrooper` (Clone Trooper)
- `cultist` (Sith Cultist)

## Notes

- Must correspond to a folder in `models/players/`.
- Use the **Asset Browser** or **Preview** button to verify.
