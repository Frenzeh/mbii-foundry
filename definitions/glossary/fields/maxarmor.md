# Max Armor

`maxarmor`

Sets the character's base maximum Armor.

> **Default:** 0

### Tips
* This is the base armor value.
* Use `MB_ATT_ARMOUR` to define the *quality* (light/medium/heavy) and effectiveness of this armor.
* Without `MB_ATT_ARMOUR` (or Cortosis/Blast Armor), this armor offers basic protection.
