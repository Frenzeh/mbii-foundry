# ❤️ Extra Health

`MB_ATT_HEALTH`

Adds bonus health **on top of** the class's base health.

> **Default Base Health:** 100 (Soldier, Jedi, etc.) to ~400 (Wookiee/SBD).

### 📊 Levels

*   **Level 1**: Adds **+?** Health.
*   **Level 2**: Adds **+?** Health.
*   **Level 3**: Adds **+?** Health.

### 💡 Tips
*   Use this to create "Tank" variants of standard classes.
*   For **Wookiees**, use `MB_ATT_WOOKIE_HEALTH` instead as it scales differently with their strength.
*   Usually better to adjust `MaxHealth` in the profile tab for precise control (e.g. set to 150), but this attribute is useful for quick preset buffs.
