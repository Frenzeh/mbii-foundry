# Fast Hacking

`CFL_FASTHACKING`

Master slicer proficiency.

### Mechanics
* **Speed:** Hacks usable objects (panels, consoles) **4x Faster**.
* **Objective:** Critical for Siege/Objective modes where time is key.

### Usage
* Tech specialists like **R2-D2**, **Imperial Workers**, or **Slicers**.
* Combine with a weak weapon loadout to create a pure "Objective Runner" class.
