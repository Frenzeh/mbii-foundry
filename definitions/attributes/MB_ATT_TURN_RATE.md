# Turn Rate

`MB_ATT_TURN_RATE`

> Engine knob for view-yaw turn limit.

## What it does

Caps how fast the player's view can yaw — engine-level tuning that's rarely picked directly. Used on tank-flavor classes (Droideka in Ball mode, AT-ST drivers) where slow turn is part of the design.

## Per level

- **Level 1** — slow yaw, sluggish view.
- **Level 2** — moderate yaw cap.
- **Level 3** — full / near-no cap.

## Notes

- Categorized as Advanced — collapsed by default in the editor.
- Player-facing builds rarely need this; usually a class-level fixed value.
- Set 0 to disable the cap entirely.
