# ❌ No Pickups

`CFL_NOPICKUPS`

Prevents the character from picking up dropped weapons.

### ⚙️ Mechanics
*   **Scavenging:** **Disabled**. You can only use what you spawn with.

### 💡 Usage
*   **Droids:** SBDs/Droidekas naturally have this.
*   **Balance:** Prevents a weak class from picking up a dropped Rocket Launcher or Concussion Rifle.
