# Top Speed

`speed`

> Maximum forward velocity of the vehicle.

## What it does

Maximum forward velocity of the vehicle.

**Default:** Varies (Speeder Bike ~800+)

## Valid values

- **Units:** Game Units per Second.
- **Comparison:** Player run speed is ~250. 500 is fast. 1000 is extremely fast.

## Notes

- Too fast = Uncontrollable on small maps.
