# Attackers

`attackers`

> Designates if this team is the attacking side (for UI/Timer logic).

## What it does

Designates if this team is the attacking side (for UI/Timer logic).

**Default:** 0 (Defender)

## Valid values

- **1:** Attackers.
- **0:** Defenders.

## Notes

- If time runs out, the **Defenders** usually win by default.
