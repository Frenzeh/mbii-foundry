# Sentry Gun

`EAS_HI_SENTRY_GUN`

> Drops an automated turret at the user's feet.

## What it does

Spawns a deployable Sentry that fires at enemy LoS targets within range. Same effect as the `HI_SENTRY_GUN` holdable, but bound to a special key so it can be deployed without weapon swap.

## Notes

- Inventory-shared with the holdable; each placement consumes one charge.
- Friendly-fire safe — Sentry will not target same-team players.
- Destructible; place behind cover or near chokes for maximum uptime.
