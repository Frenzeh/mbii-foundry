# Mind Trick

`MB_ATT_FP_TELEPATHY`

Clouds the minds of enemies, making the user invisible or creating confusion.

> **Default:** 0 FP Drain (Cannot trick).

### Levels
* **Level 1**: **20 FP** per second. Invisible on Radar. Visible to the eye but translucent.
* **Level 2**: **30 FP** per second. More translucent. Can distract enemies.
* **Level 3**: **40 FP** per second. **Invisibility**: Almost completely invisible.

### Tips
* Great for infiltration.
* Jedi with "Sense" (`MB_ATT_FP_SEE`) can see through disguises.
