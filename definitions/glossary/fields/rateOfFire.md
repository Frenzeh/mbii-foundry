# 🔫 Rate of Fire

`rateOfFire`

Multiplier for weapon firing speed.

> **Default:** 1.0 (Normal Speed)

### ⚙️ Mechanics
*   **Lower is Faster:** Values < 1.0 increase fire rate (e.g., 0.8 = 20% faster).
*   **Higher is Slower:** Values > 1.0 decrease fire rate (e.g., 2.0 = half speed).

### 💡 Tips
*   Careful with low values; can make weapons overpowered instantly.
