# primFireEnabled / altFireEnabled

`primFireEnabled 0`
`altFireEnabled 1`

> Toggle primary or alt fire entirely.

## What it does

`primFireEnabled 0` disables the primary fire button for this weapon on this class. `altFireEnabled 0` disables alt-fire. Default for both is enabled.

## Valid values

`0` (disabled) or `1` (enabled).

## Notes

- Useful for class-specific "alt-only" or "prim-only" reskins of a shared weapon.
- Disabling both makes the weapon non-functional — typically you do this only for a placeholder slot.
