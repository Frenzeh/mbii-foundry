# Move Speed Scale (Saber)

`moveSpeedScale`

> Multiplies the player's movement speed while this saber is equipped.

## What it does

Multiplies the player's movement speed while this saber is equipped.

**Default:** 1.0

## Valid values

- **1.0:** No change.
- **1.1:** +10% Speed.
- **0.9:** -10% Speed.

## Notes

- Useful for custom lightsabers that are unusually heavy (`0.8`) or very light (`1.1`).
