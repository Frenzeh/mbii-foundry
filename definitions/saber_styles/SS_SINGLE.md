# Single

`SS_SINGLE`

> One-handed saber stance — generic baseline.

## What it does

The plain, single-bladed stance used as a fallback when no specific style is selected. Not a duelist style on its own — it's the catch-all. Sabers tagged `SABER_SINGLE` can host any of the per-style purchases (`SS_BLUESTYLE`, `SS_REDSTYLE`, etc.).

## Notes

- Most custom sabers use this `saberType` and rely on style attributes (Fast/Medium/Strong) for actual combat behavior.
- Switching to two single sabers makes you Dual; the single saberType doesn't lock you into one.
- Use this as the default `saberType` for any one-handed hilt definition.
