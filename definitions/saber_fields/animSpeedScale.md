# Anim Speed Scale (Saber)

`animSpeedScale`

> Multiplies the animation speed of this saber's attacks.

## What it does

Multiplies the animation speed of this saber's attacks.

**Default:** 1.0

## Valid values

- **1.0:** Normal animation speed.
- **1.1:** +10% Faster animations.
- **0.9:** -10% Slower animations.

## Notes

- Can make an otherwise slow saber style feel much faster, or a fast one feel sluggish.
